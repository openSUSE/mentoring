Flip bits, not burgers!


Google Summer of Code is a global program that offers students stipends to write code for open source projects. openSUSE takes part in Google Summer of Code since 2006 and helped more than 50 students to start with open source!


Why should I join?

  • Because it's fun! And our openSUSE mantra is always have a lot of fun!
  • You will join an international community of awesome developers!
  • You will learn a lot!
  • An opportunity to do work related to your academic pursuits!
  • See how real world software development works!
  • You'll get a t-shirt!

For more information about the program and who is eligible please visit the official Google Summer of Code website!
