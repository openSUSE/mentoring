Flip bits, not burgers!


Google Summer of Code is a global program that offers students a stipend to write code for open source projects. openSUSE has participated in Google Summer of Code since 2006 and has helped more than 50 students to get started with open source!


Why should I join?

  • Because it's fun! Just like our openSUSE mantra - always have a lot of fun!
  • You will join an international community of awesome and talented developers!
  • You will learn a lot!
  • An opportunity to do work related to your academic pursuits!
  • See how real world software development works!
  • You'll get a t-shirt!

For more information, about the program and who is eligible, please visit the official Google Summer of Code website!

How does it work?

Check if you're eligible for GSoC on the official Google Summer of Code website!

Select one of our mentoring projects or propose one to our Administration team!

Get in touch with the mentor and follow the get started instructions in the project description!

Write an awesome project proposal and submit it to Google Melange!
